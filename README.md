# 💫 About Me

## Hi there! 👋 I'm a Software Engineer and passionate Full-Stack Developer based in Cameroon.

🛠️ **Tech Stack:** TypeScript, JavaScript, React, Next.js, NestJS, AdonisJS, React Native, PostgreSQL, Docker, Python, SQL.
🚀 I build scalable web applications and AI-powered solutions, with a strong focus on clean architecture, user experience, and performance.
📚 I created **Hekolearn**, an EdTech platform designed to improve access to quality educational resources and personalized home tutoring.
🤖 I'm passionate about Artificial Intelligence and have developed **DocMind**, a secure conversational assistant powered by Retrieval-Augmented Generation (RAG) to simplify access to enterprise knowledge.
📈 I also enjoy working on data engineering and automation projects using Python, SQL, Quarto, and scheduled workflows.
🎥 I create educational content and technical tutorials using OBS Studio and Kdenlive to help developers and students learn new technologies.
🌍 I believe technology, open source, and education can drive meaningful change across Africa.

**Let's connect and build impactful solutions together! 🚀**
